Sistema completo de agendamento de quadras esportivas com app mobile em **React Native (Expo)** e backend em **Node.js + TypeORM + SQLite**.

| Tela           | Descrição                                             |
| -------------- | ----------------------------------------------------- |
| **Home**       | Grid de esportes com imagens de capa                  |
| **Quadras**    | Lista de quadras da modalidade com carrossel de fotos |
| **Horários**   | Calendário + grade de slots disponíveis               |
| **Checkout**   | Resumo da reserva antes de confirmar                  |
| **Sucesso**    | Confirmação após reserva                              |
| **Meus Jogos** | Histórico e próximos agendamentos                     |
| **Perfil**     | Dados cadastrais e logout                             |

### Admin (login com is_admin = true)

| Tela                   | Descrição                                         |
| ---------------------- | ------------------------------------------------- |
| **Linha do Tempo**     | Visualização diária das quadras + bloqueio manual |
| **Gerenciar Quadras**  | CRUD de quadras com fotos e diferenciais          |
| **Gerenciar Esportes** | CRUD de modalidades com foto de capa              |
| **Ajustes da Arena**   | Dados institucionais e horário de funcionamento   |
